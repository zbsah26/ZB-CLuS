## 📊 Feature Quick Access

### Mathematical Solver
**Location**: Top navigation → Calculators → Math Solver
**How to use**:
1. Enter expression: `sqrt(16) + 5 * 2`
2. Click "Calculate"
3. Get instant result with AI explanation

**Supported Operations**:
- Basic: `+`, `-`, `*`, `/`, `^`
- Functions: `sqrt()`, `sin()`, `cos()`, `tan()`, `log()`, `ln()`, `abs()`

---

### Physics Solver
**Location**: Calculators → Physics Solver
**Tabs Available**:

#### Motion (v = u + at)
- Initial Velocity (u): meters/second
- Acceleration (a): m/s²
- Time (t): seconds
- Result: Final velocity & displacement

#### Force (F = ma)
- Mass: kilograms
- Acceleration: m/s²
- Result: Force in Newtons

#### Energy (KE = ½mv²)
- Mass: kilograms
- Velocity: m/s
- Result: Kinetic Energy in Joules

### Interest Calculator
**Location**: Calculators → Interest Calculator

#### Simple Interest (SI = P×R×T/100)
**Steps**:
1. Select country (Nepal, India, China, USA)
2. Enter Principal Amount
3. Enter Interest Rate (% per annum)
4. Enter Time (years)
5. Click "Calculate SI"

#### Compound Interest (A = P(1+r/100n)^nt)
**Steps**:
1. Select country
2. Enter Principal Amount
3. Enter Interest Rate
4. Enter Time (years)
5. Select Frequency (Annually/Semi-annually/Quarterly/Monthly)
6. Click "Calculate CI"

### Enable JavaScript
ZB CLuS requires JavaScript to be enabled.

- **Firefox**: about:config → javascript.enabled ✓
- **Safari**: Preferences → Security → JavaScript ✓

- **Chrome**: Ctrl+Shift+Delete
- **Firefox**: Ctrl+Shift+Delete
- **Safari**: Develop → Empty Web Storage

## 🔧 Running Locally

### Option 1: Direct Open (Easiest)
1. Double-click `index.html`
2. Opens in your default browser

### Option 2: Python Server
```bash
# In the folder containing the files
python -m http.server 8000
# Open http://localhost:8000 in browser
```

### Option 3: Node.js Server
```bash
# Install http-server globally
npm install -g http-server

# Run server
http-server

# Open http://localhost:8080 in browser
```

### Option 4: PHP Server
```bash
# In the folder
php -S localhost:8000
# Open http://localhost:8000 in browser
```
